# 前后端常用命令

## 前端

```bash
npm cache verify
rmdir /s /q node_modules
del package-lock.json
npm install --verbose
npm run typecheck
npm run build 
npm run dev
```

## 后端

```bash
python -m pip install -e ".[dev]"
python -m alembic upgrade head
python -m uvicorn app.main:app --reload
python -m pytest -q
```
